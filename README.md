Данный репозиторий хранит выполненные практики 1-6.
Для перехода к какой-либо практики следует обратиться к соответствующему коммиту.
Проект использует Node.js, Express и React.
Документация по API реализована с использованием Swagger.
Перед использованием установить зависимости с помощью `npm install`.
Для запуска используется `node server.js` для бекенда и `npm run dev` или `npm run build` для фронтенда.
Бекенд выполнен по CRUD методологии, логирует все запросы, валидирует их.
Фронтенд отображает простую страницу где отображается всё необходимое.
